# Check executable(s)

Check that each executable works. Support function for
[find_executables_remote](https://rajlabmssm.github.io/echoconda/reference/find_executables_remote.md).

## Usage

``` r
check_executables_remote(path, verbose = TRUE)
```

## Arguments

- path:

  Path or named list of paths to executable(s).

- verbose:

  Print messages.

## Value

Logical list.
