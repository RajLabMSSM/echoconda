# Print the currently activated conda env

[py_discover_config](https://rstudio.github.io/reticulate/reference/py_discover_config.html)
"This function enables callers to check which versions of Python will be
discovered on a system as well as which one will be chosen for use with
reticulate."

## Usage

``` r
which_env(verbose = TRUE)
```

## Arguments

- verbose:

  Print messages.
