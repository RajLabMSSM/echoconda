# Create conda env: basilisk

Create conda env using one of several methods.

## Usage

``` r
create_env_basilisk(
  yaml_path,
  pkgs,
  sep = "=",
  use_subversion = TRUE,
  use_nodefaults = TRUE,
  start = TRUE
)
```

## Source

[GitHub repo](https://github.com/LTLA/basilisk)

Bioconductor
([doi:10.18129/B9.bioc.basilisk](https://doi.org/10.18129/B9.bioc.basilisk)
)

## Arguments

- yaml_path:

  Path to local or remote yaml file with conda build specifications.

- pkgs:

  data.table with the columns:

  "package" :

  :   Package names.

  "version" :

  :   Package versions.

  "channel" :

  :   Channel where the package was installed from.

  This table can be generated by
  [env_to_yaml](https://rajlabmssm.github.io/echoconda/reference/env_to_yaml.md)
  function.

- use_subversion:

  Specify the specific package subversion.

- start:

  Execute
  [basiliskStart](https://rdrr.io/pkg/basilisk/man/basiliskStart.html)
  to build the actual environment.

## Value

An object of class
[BasiliskEnvironment](https://rdrr.io/pkg/basilisk/man/BasiliskEnvironment-class.html).
