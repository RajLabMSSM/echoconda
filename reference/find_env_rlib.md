# Find the R library for a specific env

Find the R library for a specific env

## Usage

``` r
find_env_rlib(conda_env = "echoR_mini", suffix = "lib/R/library/")
```

## Arguments

- conda_env:

  Conda environment name.

- suffix:

  Suffixes to use when searching for R library.
